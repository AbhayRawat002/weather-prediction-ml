# Weather Prediction Using Machine Learning

## About the Project

This project is a simple machine learning-based weather prediction system built using historical weather data from Dehradun, India.

The goal of the project was to explore the complete machine learning workflow, including:

- Data collection
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Model training
- Weather prediction

This project was created as a learning exercise to gain practical experience with Python, Pandas, Scikit-Learn, and machine learning concepts.

---

## Dataset

The dataset contains historical weather records for Dehradun collected across multiple years.

Files used:

- dehradun 2015-01-01 to 2015-12-31.csv
- dehradun 2016-01-01 to 2017-12-31.csv
- dehradun 2018-01-01 to 2019-12-31.csv
- dehradun 2020-01-01 to 2021-12-31.csv
- dehradun 2022-01-01 to 2023-12-31.csv
- dehradun 2024-01-01 to 2024-01-31.csv

The data includes weather-related attributes such as temperature, humidity, precipitation, wind conditions, and other meteorological observations.

## Project Workflow

### 1. Data Collection

Historical weather data was collected and combined from multiple CSV files covering different years.

### 2. Data Cleaning

- Handling missing values
- Removing inconsistencies
- Formatting columns
- Preparing data for analysis

### 3. Exploratory Data Analysis

Basic analysis and visualizations were performed to understand:

- Temperature trends
- Seasonal variations
- Rainfall patterns
- Weather distribution

### 4. Feature Preparation

Relevant weather features were selected and transformed into a format suitable for machine learning models.

### 5. Model Training

Machine learning algorithms were trained using historical weather data to learn patterns and relationships between weather variables.

### 6. Prediction

The trained model was used to predict future weather conditions based on input parameters.

## Results

The project demonstrates how machine learning can be applied to historical weather data for prediction tasks.

While the model is intended primarily for learning and experimentation, it provides useful insights into weather trends and forecasting techniques.
